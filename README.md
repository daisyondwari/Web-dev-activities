# Web Development Lab Activities

This repository contains activities for **HTML**, **CSS**, and **JavaScript**.  
Each activity corresponds to exercises in your lab manual. Follow the sequence carefully.

---

## 🔹 How to Use This Repository

1. **Clone or Download**
   - Clone: `git clone https://github.com/yourusername/web-dev-lab-activities.git`
   - Or download as ZIP and extract.

2. **Navigate to the Correct Folder**
   - Go to `html/`, `css/`, or `javascript/` depending on the lab manual section.

3. **Create Your Activity File**
   - For each lab activity, create a new file named:
   - `activity1.html`, `activity2.html`, etc. (for HTML)
   - `activity1.css`, `activity2.css`, etc. (for CSS)
   - `activity1.js`, `activity2.js`, etc. (for JavaScript)

4. **Follow Lab Manual Instructions**
   - Write your code inside the file.
   - Save and test in your browser or editor.

5. **Commit and Push (if using GitHub)**
   ```bash
   git add .
   git commit -m "Completed HTML Activity 1"
   git push origin main
